## Setup Steps :-

```bash
git clone https://github.com/cfg2india2021/team-52
git checkout develop
git pull

npm i nodemon -g
npm install express
npm install body-parser
npm install --save firebase
yarn
cd client
yarn
```

## To run the App :-
```bash
yarn dev
```

Go to http://localhost:3000
